Pseudo-code: Attendance Marker
1. Define 2D array attendance[students][days]

2. Operations:
    - Mark attendance: attendance[i][j] = 'P' or 'A'
    - Update attendance: modify cell value
    - Generate report: count 'P' for each student

3. Example:
    Student1 week1: P,P,A,P,P,P,A
    Update day3 A->P
    Report: Student1 present 26 days
