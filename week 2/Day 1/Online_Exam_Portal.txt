Pseudo-code: Online Exam Portal
1. Define class User
    attributes: username, password
    method: login()

2. Define class Student extends User
    method: login()
        check if student is registered
        check if enrolled in course
        if true -> grant access
        else -> deny access

3. Define class Admin extends User
    method: login()
        check if credentials valid
        check if role is admin
        if true -> allow exam management
        else -> deny access

4. Example Flow:
    input username, password
    if student -> validate enrollment -> show exams
    if admin -> validate privileges -> show exam tools
