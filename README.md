# Java Assignment – Advanced Concepts

## 📌 Overview
This project is a **Java 8 assignment** demonstrating the use of **regular expressions, functional interfaces, annotations, and modern Java features**.  
The goal is to apply Java concepts in real-world coding scenarios and write clean, efficient, and reusable code.

---

## 🚀 Features
- **Regular Expressions (Regex):**
  - Pattern matching for validation (e.g., email, phone number, password).
  - String manipulation using `Pattern` and `Matcher`.

- **Functional Interfaces & Lambda Expressions:**
  - Custom functional interfaces.
  - Use of built-in interfaces like `Predicate`, `Consumer`, and `Function`.
  - Cleaner and more concise code with lambda expressions.

- **Annotations:**
  - Custom annotation creation.
  - Use of `@Override`, `@FunctionalInterface`, and custom markers for metadata.

- **Java 8 Features:**
  - Stream API for collection processing.
  - Default and static methods in interfaces.
  - Method references for cleaner code.

---

## 🛠️ Technologies Used
- **Java SE 8**
- **JDK 1.8+**
- IDE: IntelliJ IDEA / Eclipse / VS Code (any)
