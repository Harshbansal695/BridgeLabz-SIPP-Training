# Java Assignment – From Basics to Advanced

## 📌 Overview
This project is developed as part of a **Java assignment**, covering concepts from **core Java fundamentals** to **advanced Java 8 features**.  
The purpose is to demonstrate problem-solving skills, object-oriented principles, and the usage of **modern Java features like Regex, Functional Interfaces, Streams, and Annotations**.

---

## 🧑‍💻 Topics Covered

### 1️⃣ Java Basics
- **Variables, Data Types, Operators**
- **Control Statements** (`if-else`, `switch`, loops)
- **Arrays and Strings**
- **Methods & Recursion**

### 2️⃣ Object-Oriented Programming (OOP)
- **Classes and Objects**
- **Encapsulation** (getters & setters)
- **Inheritance** (single & multilevel)
- **Polymorphism** (compile-time & runtime)
- **Abstraction** (abstract classes & interfaces)

### 3️⃣ Exception Handling
- `try-catch-finally`
- `throw` and `throws`
- Custom exceptions

### 4️⃣ Collections Framework
- **List, Set, Map**
- Iterators and Generics
- Sorting with **Comparator** and **Comparable**

### 5️⃣ Java 8 Features (Advanced)
- **Lambda Expressions** for concise code
- **Functional Interfaces** (`Predicate`, `Consumer`, `Supplier`, `Function`)
- **Streams API** for working with collections
- **Method References** and **Default Methods in Interfaces**

### 6️⃣ Regular Expressions (Regex)
- Pattern matching using `Pattern` and `Matcher`
- Validations (Email, Phone Number, Password strength, etc.)
- String searching and replacement

### 7️⃣ Annotations
- Built-in Annotations: `@Override`, `@Deprecated`, `@FunctionalInterface`
- Custom Annotations with `@interface`
- Reflection API to process annotations

---

## 🚀 Features of the Project
- Demonstrates **step-by-step growth** from basic Java to advanced features.
- Covers **real-world examples** (like email validation with regex).
- Includes **functional programming approach** with Java 8.
- Applies **annotations** for metadata and code documentation.
